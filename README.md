# Manus Agency Skills

A collection of 72 AI Agent Skills from the [agency-agents](https://github.com/msitarzewski/agency-agents) repository, properly formatted for direct import into Manus.

## About

These skills are extracted from the comprehensive agency-agents repository and restructured to follow the Manus Skill format specification. Each skill is in its own folder with a `SKILL.md` file.

## How to Use

1. In Manus, go to **Skills** → **+ Add** → **Import from GitHub**
2. Paste an individual skill folder URL like: `https://github.com/WebWakaHub/manus-agency-skills/tree/main/engineering-ai-engineer`
